# Archivos de la materia de Criptografía y Seguridad Informática

# Contenido

## Ciphers

- playfair
- caesar
- vigenere
- hill
- turning grille

> To run the ciphers use python -i ./\*\*/cipher_name.py
> Then run help(cipher_name) to see how to use the cipher
> Run cipher_name(...) to encrypt or decrypt a message

## DES and AES

Create a venv using `python3 -m venv .venv` and activate it
Then install the dependencies using `pip install -r requirements.txt`
It will install pyDes and pyaes

Needs an image for encryption and decryption

> Run python ./\*\*/DES.py or ./\*\*/AES.py to run the DES or AES ciphers
